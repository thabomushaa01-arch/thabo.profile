BIT315 S2 2025 - WEB DEVELOPMENT ASSIGNMENT
Student Portfolio Website

==============================================
QUICK START GUIDE
==============================================

1. LOGIN CREDENTIALS:
   - Username: Any 4+ characters (e.g., "john", "student", "admin")
   - Password: Any 6+ characters (e.g., "password", "123456")

   Note: This is front-end validation only (no backend)

2. GETTING IMAGES:

   OPTION 1 - Use Placeholder Generator (Quick):
   - Open "create-placeholders.html" in your browser
   - Click each download button to save placeholder images
   - Save all images to the "images/" folder

   OPTION 2 - Download Free HD Images (Recommended):
   Go to these FREE stock photo websites:

   - Unsplash: https://unsplash.com
   - Pexels: https://pexels.com
   - Pixabay: https://pixabay.com

   Download these images and save to images/ folder:

   Required Images:
   ├── images/
   │   ├── profile.jpg (400x400) - Professional headshot
   │   ├── slide1.jpg (800x400) - Campus/building
   │   ├── slide2.jpg (800x400) - Library
   │   ├── slide3.jpg (800x400) - Computer lab
   │   ├── slide4.jpg (800x400) - Graduation
   │   ├── slide5.jpg (800x400) - Students studying
   │   ├── coding.jpg (400x300) - Person coding
   │   ├── reading.jpg (400x300) - Person reading
   │   ├── music.jpg (400x300) - Musical instrument
   │   └── career.jpg (400x300) - Professional workspace

   OPTION 3 - Use Your Own Photos:
   - Use your actual photos
   - Resize to recommended dimensions
   - Save to images/ folder

3. VIDEOS & AUDIO (Optional for now):

   The website will work without these, but for full marks you need:

   videos/ folder:
   - tutorial.mp4 - Screen recording of website navigation
   - inspiration.mp4 - Any motivational/tech video

   audio/ folder:
   - study-music.mp3 - Any instrumental music

   Free resources:
   - Videos: Create screen recording or use YouTube (with permission)
   - Audio: Free music at freemusicarchive.org or bensound.com

4. TESTING:
   - Open index.html in your browser
   - Login with any valid credentials
   - Navigate through all pages
   - Test on mobile (responsive design)

5. SUBMISSION:
   - Zip the entire BIT315_Portfolio folder
   - Ensure all HTML, CSS, JS files are included
   - Include all images/media used
   - Website must work offline
